About this project

I am happy to see you here

Generally

I am currently studying at University of Miskolc and working on my thesis. This project is part of my thesis. My goal to build different machine learning models which can generate personally predicts for different users based on the user previous cycling data and compare these model based on their performance and efficiency.

How can you help me

To build a good model I need a lot of data from different age groups like thousands of magnitude. I planning to use the Strava application for collect data but of course for this I need users who give permission for my app to read their data. Only statistical data are concerned none of the personal info will be used (like name, e-mail address etc) and all the activities data will be anonymised and will be deleted after the model learnt from it. So if you would like to help my work with sharing your cycling statistic please connect to your Strava account. Of course you can revoke the permission anytime under your Strava accounts' settings My Apps menu.
